# Ecommerce website

Implementing an E-commerce Website Using Django.


## Getting Start
- Clone Project
  ```
    git clone https://github.com/pong1013/ecommerce-django.git
  ```

- Run Project
    ```
    cd ecom
    python manage.py runserver
    ```
